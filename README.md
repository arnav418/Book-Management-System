# Books Management System

A modern Library Management System built using React and Vite.

## Features

- Add Books
- Edit Books
- Delete Books
- Search Books
- Dynamic Genre Filter
- Dark Mode
- Responsive Design
- Toast Notifications
- Modal Forms

## Tech Stack

- React
- Vite
- CSS3
- Axios
- JSON Server / API

## Installation

```bash
npm install
npm run dev
```
